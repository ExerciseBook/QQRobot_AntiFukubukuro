# QQ福袋撤回 for CoolQ
![title](image/title.jpg)

## 概述
- 这个QQ福袋太烦人了。我受不了这委屈.jpg

## 特性
- 自动撤回福袋
- 可以分群开关
- 配置简单，下载即用

## 命令
- 开启自动撤回
	- FUKUBUKURO-ON
	- 撤回福袋
	- 福袋撤回
- 关闭自动撤回
	- FUKUBUKURO-OFF
	- 放行福袋
	- 福袋放行
